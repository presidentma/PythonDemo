            var target = $(this.hash);
            target = target.length ? target : $('[name=' + this.hash.slice(1) + ']');

                $('html,body').animate({
                    scrollTop: target.offset().top - xbFloatAdjustments //offsets for fixed header

                return false;

                scrollTop: target.offset().top - xbFloatAdjustments //offset height of header here too.
            }, 1);
            return false;
        }
    }
});

					<b>关于如何在目录提交您的网站</b><br />
<br />
<b>关于如何在目录提交您的网站</b><br />
<ul>
<li>我可以找到很多适合于我的网站收录的目录 - 是否应该提交到所有的目录？</li>
</ul>不，请不要这样做。通常只有一个目录最合适收录您的网站，当然也有例外。如果你无法确认哪个目录最合适，可以试着发在论坛，请教编辑员的建议。我们愿意给予提交者一些建议，这远比我们花更多的时间处理在不同目录的网站简单地多。<ul>
<li>我所提交的目录没有编辑员 - 我所提交的网站还能被审核吗？</li>
</ul>你所认为目录无编辑员是因为看到的目录下方没有列名的编辑员。但这不意味着该目录没有编辑员管理。任何更高权限的编辑员都可以对以下的目录进行编辑与管理。除此之外，有两类编辑员（Editalls &amp; Metas）可以编辑任何一个目录。所以，总会有人处理每个提交的网站。<ul>
<li>在我再次提交网站前，我应该等待多久呢？</li>
</ul>当一个网站提交到ODP，它则在目录的待审核队列等待编辑员审核。提交的网站不会过期。所以一般采取的方式是请提交您的网站到目录，然后耐心等待至少一个月，如果一个月后网站没有被发布，你可以选择再次提交，但请注意后者将覆盖之前所提交的网站。编辑员有需要方式审核网站，而通常大部分会选择按照时间先后的顺序，如果是这样的话，多次提交网站只会使得您的网站需要等待更长的时间。<ul>
<li>为什么我不能使用ODP搜索找到我的网站？</li>
</ul>ODP是一个目录，不是搜索引擎。ODP的搜索功能目的是作为一个方便的工具帮助寻找到具体专题的某个目录，而不是网站。它的数据是根据标题与描述，而不是网站具体内容。它没有使用关键词列表，不提供任何方式的排名分析，对于普通用户它不是作为一个搜索引擎。<br />
如果需要搜索到具体网站，那么最好的方式是将网址去除www进行搜索。举例，如果你需要寻找<a href="http://www.example.com" target="_blank" class="externalLink ProxyLink" data-proxy-href="proxy.php?link=http%3A%2F%2Fwww.example.com&amp;hash=5c0d7893096d210fbebbfe8238d56473">http://www.example.com</a> 其中包括一些具体的子页面 (在ODP目录所谓的 “深层链接” )，试着搜索“example.com”。搜索数据通常有滞后性，至少可能无法搜索到最近一周甚至更长时间所更新的网站。<br />
<br />
由于这些因素在，应该说没有一种更可靠的方式查询一个具体的网站是否被加入ODP。<ul>
<li>我能否提交我的网站到专题目录和相应的地区目录呢？</li>
</ul>如果网站是关于地区目录的，好，那么，举例来说，一个网站如果更合适于或对于用户而言更突出地区特色的，那么，收录在地区应该说更合适。<br />
一些网站只列在地区目录，有一些从来不列在地区目录，所以总是有很多例外。如果你认为网站合适于同时加入两个目录，那么就请同时提交他们 - 你不会为此而受处罚。<ul>
<li>需要等待多久我的网站才能被审核？</li>
</ul>这个时间（从提交网站到编辑员审核网站）可能需要二分钟，也可能需要二年。这取决于ODP义务的本质 - 编辑员义务为目录工作的空闲时间。当然审核网站仅仅是其中很多任务中的一项，对于很多编辑员这并不是首要他们考虑的。<br />
审核网站并发布仅仅是对于丰富ODP网站资源的一个方式 - 我们鼓励编辑员自己寻找优秀的网站并主动为目录加入。<ul>
<li>为什么只列我的竞争者的网站，而我的网站还没有被加入？</li>
</ul>通常我们按照先后顺序审核每个独立的网站，这也是很多编辑员所做的事情。所以在你看来会有可能竞争者的网站先被加入。<ul>
<li>那么多糟糕的网站已经被收录，为何我的网站没有被收录？</li>
</ul>正如他们所说的，美丽来自于目睹者的眼中(情人眼里出西施)。 首先不同的人对于网站有着不同的看法。这也就是我们需要根据ODP编辑指南，要求我们的编辑员按照指南进行合理的编辑。<br />
其次，标准的改变 - 数年前，当ODP还是未成熟的目录，其特殊性致使需要越快越好的加入更多的网站。如今，我们把重点转移到提高网站的质量上来。所以可能现在被发布的网站有一些在现在看并不是那么高质量。在这种情况下，如果当编辑员发现他们，我们会将其删除。如果你认为一些是滥用目录导致的问题，请是滥用目录表格的方式，或联系一个 meta编辑员，向他们描述滥用目录的情况。<br />
<br />
第三，网站的改变 - 在很多情况可能网站站长改变了网站的内容，使其网站内容完全与先前内容不相同。如果你发现这样的网站，你如果希望帮助ODP成为一个更好的目录，那么建议通过滥用目录表格向我们汇报这些问题。<br />
<br />
最后，随着ODP的逐渐发展，我们对于网站需要唯一独特的要求越来越高。如果我们有这样的想法，“好吧，既然这个网站没比其他网站更糟糕，所以我打算加入它”，这样如何改善目录呢？如果你希望你的网站被列在一个目录中，它应该比其他最好的网站有着更丰富独特的内容。不要做第二个“最糟糕的”网站，目标是要做最好的网站。<br />
